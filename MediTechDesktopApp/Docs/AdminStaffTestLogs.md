### AS-TC01 — Add Admin Staff

**Feature:** Add Admin Staff  
**Status:** Working as expected

<details>
<summary>🔍 Steps</summary>

1. Launch app, log in.  
2. Click **AdminStaff** in left navigation.  
3. Click **New**.  
4. Enter:  
   - First Name = “Jane”  
   - Last Name  = “Doe”  
   - Role       = “Billing Clerk”  
   - Phone      = “555-1234”  
   - Email      = “jdoe@example.com”  
5. Click **Save**.
